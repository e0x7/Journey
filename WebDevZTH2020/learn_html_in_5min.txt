### This is the rule that every html file must follow (4.1)
```
<!DOCTYPE html>
<html>
  <head>
    <title></title>
  </head>
  <body>

  </body>
</html>
```

### Play around with these tags and understand them.

<html>
<head>
<title>
<body>
headings (h1, h2, h3, h4, h5, h6)
paragraph <p>
bold <strong>, italic <em>
ordered list <ol>, unordered list <ul>, list item<li>
break <br>, horizontal rule <hr>
image <img> and attributes: src, width, height
anchor <a href="">
